# Changelog

All notable changes to `@citypay/elements-react` will be documented in this file.

This project follows semantic versioning:

- patch releases contain backwards-compatible fixes
- minor releases add backwards-compatible components, hooks, options, or supported flows
- major releases contain breaking API or runtime behavior changes

## 0.1.0

- Initial React wrapper for CityPay Elements.
- Added `CityPayProvider`, `CardElement`, `CardFieldsProvider`, `CardFields`, and `ApplepayElement`.
- Added ESM, CommonJS, and TypeScript declaration outputs.
